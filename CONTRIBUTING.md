# Contributing

This repository documents a working method (the CA6SNT Agent Studio). Contributions, questions and adaptations are welcome.

- **Issues** — open one to discuss an idea, a gap in the docs, or an adaptation of the method to another stack.
- **Pull requests** — keep changes focused. Docs follow the studio's own principle P-09: document reality, don't inflate.
- **Scope** — the studio is project-agnostic (P-10). Please keep contributed material stack-neutral; project-specific examples belong in your own fork.

By contributing you agree your contributions are licensed under the repository's [MIT License](LICENSE).
